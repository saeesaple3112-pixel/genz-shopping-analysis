# GenZ Shopping Analysis Dashboard

An interactive data analytics dashboard built using R and Shiny to analyze Gen Z shopping behavior and purchasing patterns.

## Features

- Data cleaning and preprocessing using CSV dataset
- Statistical analysis including:
  - Correlation and Regression
  - ANOVA
  - Chi-square test
  - t-test
  - Poisson distribution
- Interactive visualizations using Plotly
- Multi-tab dashboard for:
  - Purchase Influences
  - Sales Analysis
  - Consumer Behavior
  - Market Segmentation
  - Buying Decision Analysis

## Tech Stack

- R
- Shiny
- shinydashboard
- ggplot2
- plotly

## Dataset

The dataset (`customers.csv`) contains customer purchasing behavior used for analysis and visualization.
